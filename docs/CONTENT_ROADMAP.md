## How this was produced

Generated by a fan-out design workflow (one grounded agent per area → synthesis). Every section is a
*plan*, not implementation: sections mark invented coordinates as `PROPOSED (needs builder confirm)`
and end with **"Implementation notes / future-me hooks"** (exact files, pipeline steps, patterns to
copy). Verify a section's file-level claims before building from it.
